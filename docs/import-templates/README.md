# 會友名單匯入範本（中文欄位）

> 給幹事整理名單用。兩份範本，依需求擇一或並用。
> ⚠️ **重要**：目前系統的匯入器**只認英文欄位、且每列必填 P2 事由**。
> 這兩份**中文 header 範本**要能「直接上傳匯入」，需先完成 triage 的兩個功能
> （中文 header 對照層＋簡易全體匯入）。在那之前，本範本可先用來**收集/整理名單**，
> 功能上線後即可直接匯入。範例資料皆為**虛構**，非真實個資。

---

## 1. 會友簡易名單範本.csv —— 全體會友（多數 P3）

最精簡、最好「看清楚全體」的格式。

| 欄位 | 說明 | 必填 |
|------|------|------|
| 姓名 | 會友姓名 | ✅ |
| 手機 | 手機號碼（同一人多車請填同號碼） | ✅ |
| 車牌 | 車牌（自動轉大寫、去空白/橫線比對） | ✅ |
| 優先序 | `P3`＝一般會友；`P2`＝需資格（要填 P2事由）；`P1`＝全職同工（另管） | ✅ |
| P2事由 | 僅 P2 需填：行動不便／短期不便／幼兒同行／孕婦／長者同行 | 選填 |
| 備註 | 自由文字 | 選填 |

**一人多車**：同手機、同姓名，分成多列（每台車一列）。

> ⚠️ **手機開頭 0 會被 Excel 吃掉**：用 Excel／Numbers 打開或編輯本 CSV 時，`0912345678` 會被當數字自動變成 `912345678`（掉開頭 0）。
> 因手機是會友身分主鍵，掉 0 會導致匯入失敗或對不上。
> **做法**：編輯前先把「手機」整欄設為**文字格式**（Excel：選欄→儲存格格式→文字），或改用純文字編輯器／Google 試算表（匯入時指定該欄為文字）。
> （匯入器端亦已列 backlog：自動把「9 碼開頭 9」補回 `09`，見 feature-triage #22。）

## 2. P2資格申請範本.csv —— P2 資格申請（沿用現有紙本申請表）

欄位較多，對應紙本「停車申請表」。**申請原因用代碼**：

| 代碼 | 意義 | 資格效期 | 必附 |
|------|------|----------|------|
| 1 | 行動不便（長期／身障） | 永久 | 行動不便者姓名 |
| 2 | 行動不便（短期） | 自申請日 +6 個月 | 行動不便者姓名 |
| 3 | 幼兒同行（未滿 5 歲）**或**孕婦 | 幼兒：生日 +5 年；孕婦：另計 | 幼兒→孩童姓名+生日；孕婦→備註寫「懷孕」 |
| 4 | 長者同行（75 歲以上） | 永久 | 長者姓名 + 生日 |

- 日期格式：申請日期 `YYYY-MM-DD`；生日 `YYYY/MM/DD`。
- 幼兒最多 3 位（孩童姓名/生日 1–3）。
