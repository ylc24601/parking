📋 主日流程說明（點開看）▾
同工要動手的只有「點名」「補點名」，其餘系統自動。
  1. 到崗：打開頁面，自動列出本週該來的車。
  2. 車到・點名：打車牌後四碼 → 按「點名」→ 變已到。
  3. 時間到・自動釋出：沒來的車系統自動轉「已釋出」（同工不用動手）。
  4. 遲到・補點名：對已釋出的車按「補點名」，免記違規。
  5. 登記散客 walk-in（第一版先不做）
  6. 結束當週點名（第一版先不做）
車到就「點名」、遲到就「補點名」，其餘交給系統。
