使用方式

會友在 LINE 官方帳號點選選單，即可開啟 LIFF 網頁。不需要下載額外 App，操作完全在 LINE 內完成。

會友可以做什麼

  • 🚗 登錄多組車牌，每週選「本週開哪台」
  • 📝 申請本週停車（週一起開放，週五 18:00 截止）
  • 👨‍👩‍👧 宣告本週有年長者 / 幼兒同行（P2 優先）
  • 📊 即時查看本週狀態（已核准 / 候補 / 未申請）
  • ❌ 主日 10:30 前取消，不計違規紀錄

💚 恩慈設計亮點

取消確認頁面明確顯示「不計違規紀錄」，目的是讓會友安心取消而不是硬撐——若臨時無法出席卻因害怕被記點而不取消，反而讓候補者白等、白白浪費一個車位。

⚠️ 待同工長執確認的規則（以下為提案，尚未定案）

  • 🔑 P1 全職同工當週不到（外教會服事）
    提案：週五前於系統標記「本週不停」，P1 保留名額自動釋出給一般候補
  • 👶 P2 幼兒家庭遲到緩衝設計
    提案：10:20 LINE 提醒 P2 未到者，可回「正在路上」；P2 車位延至 10:45 才自動釋出（一般會友仍 10:30）
  • 🤰 孕婦 P2 有效期
    提案：Admin 核准後持續有效至孩子達學齡（小學入學）為止
  • ❓ P2 關懷會友：No-show 完全不計懲罰？
    目前設計：P2 不累積 penalty score，No-show 僅觸發牧養關懷聯繫。
    設計理由：P2 群體（行動不便、孕婦、幼兒同行、年長者）因身體或家庭因素本已處於較脆弱狀態，懲罰機制恐產生心理負擔，與教會牧養精神相違。建議以關懷取代管制。
    ⇒ 請長執確認是否接受此設計，或需區分 P2 原因類別分別處理。
  • 📋 其他
    P2 年長/幼兒宣告需每週勾選，還是可設「常態同行」自動套用？
    首次綁定需要哪些資料？（姓名 + 車牌 + 教會會友編號？）
